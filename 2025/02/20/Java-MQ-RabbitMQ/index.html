发表于 2025-02-20 更新于 2025-03-15 分类于 Java ， 框架与工具 本文字数： 8.6k 阅读时长 ≈ 29 分钟
MQ系列是学习消息队列中间件时记录的笔记，本文记录了RabbitMQ相关的内容。消息队列（MQ)是一种异步通信机制，允许应用程序通过存储和转发消息来解耦、缓冲流量及分发数据。它使得生产者和消费者可以独立运作，提升系统的灵活性和可靠性。RabbitMQ是一个流行的开源消息队列系统，支持多种消息传递模式，确保高效可靠的消息传递，便于构建稳健的分布式应用。
